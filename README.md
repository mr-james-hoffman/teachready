# TeachReady

TeachReady is a prototype planning assistant for primary teachers.

It helps a teacher turn a learning goal and class context into a practical lesson sequence, including explicit teaching, guided practice, checks for understanding, likely misconceptions, responsive teaching moves, support, extension and an exit ticket.

This project was built for OpenAI Build Week using Codex and GPT-5.6.

## Problem

Teachers often know what they need students to learn, but need fast, practical help turning that goal into a lesson they can teach tomorrow.

## Solution

TeachReady turns a teacher's learning goal and class context into a concise, classroom-ready teaching sequence.

## Safety and limitations

TeachReady is a planning assistant. Teachers should review and adapt all suggestions using their professional judgement, curriculum requirements, school context and student needs.
